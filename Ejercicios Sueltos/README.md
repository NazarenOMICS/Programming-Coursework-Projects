# Ejercicios de práctica – Python

Esta carpeta contiene distintos ejercicios y pequeños scripts desarrollados durante el curso de programación.  
A diferencia de los obligatorios del curso, estos archivos son prácticas individuales enfocadas en aprender y probar distintos conceptos del lenguaje Python.

El objetivo de estos ejercicios fue familiarizarse con estructuras de control, manejo de datos y herramientas básicas del lenguaje.

## Conceptos practicados

Los ejercicios incluidos exploran distintos aspectos fundamentales de Python:

**Estructuras de control**
- Condicionales (`if`, `elif`, `else`)
- Bucles (`for`, `while`)

**Estructuras de datos**
- Listas
- Diccionarios

**Manejo de errores**
- Uso de `try` y `except` para controlar errores de entrada y ejecución

**Programación orientada a objetos**
- Definición de clases
- Creación de objetos y métodos

**Manejo de archivos**
- Lectura y escritura de archivos utilizando `open()` y `with`

**Otros conceptos**
- Ordenamiento de listas
- Uso de funciones
- Validación de datos ingresados por el usuario
- Manipulación básica de secuencias y strings

## Objetivo

Estos scripts funcionan como ejemplos simples de práctica para comprender los fundamentos de programación en Python antes de desarrollar programas más complejos como los incluidos en los obligatorios del curso.
